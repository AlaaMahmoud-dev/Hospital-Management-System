<h1 align="center">🏥 Hospital Management System (HMS)</h1>
<h3 align="center">Desktop Application for Managing Patients, Doctors, and Medical Records</h3>

---

## 📌 Overview
The **Hospital Management System (HMS)** is a desktop application built using **.NET (WinForms)** designed to manage core hospital operations including patients, doctors, users, and prescriptions.

The system aims to simulate a real-world healthcare workflow starting from **patient registration** to **treatment tracking** and prescription management.

> ⚠️ Note: This project is currently **in progress** and will be extended to fully support the patient lifecycle from admission to discharge.

---

## 🚀 Key Features

### 👤 Patient Management
- Add and manage patient records
- View patient list and detailed information

---

### 💊 Prescription Management
- Create and manage prescriptions
- Link prescriptions to patients and doctors
- Track prescription details and history

---

### 👨‍⚕️ Doctor Management
- Add and manage doctors
- Store doctor-related information

---

### 👥 User Management
- Manage system users
- Control access to system features

---

## 🧠 System Workflow

1. User logs into the system  
2. Add a new patient  
3. Assign doctor and create prescription  
4. Track patient treatment and medical data  

---

## 🏗️ Architecture

The system follows a **3-Tier Architecture**:

- 🎨 Presentation Layer (WinForms UI)
- ⚙️ Business Logic Layer
- 🗄️ Data Access Layer (ADO.NET)

### ✅ Highlights:
- Separation of concerns
- Maintainable and scalable structure
- Clear organization of responsibilities

---

## 🛠️ Technologies Used

- **C# / .NET (WinForms)**
- **SQL Server**
- **ADO.NET**
- **3-Tier Architecture**

---

## ⭐ What Makes This Project Stand Out

✔️ Simulates real-world hospital operations beyond simple CRUD  
✔️ Structured using **3-Tier Architecture** for maintainability  
✔️ Designed modular **Presentation Layer** with reusable UI components  
✔️ Strong focus on organizing patient and medical data  
✔️ Database-driven system with clear entity relationships
